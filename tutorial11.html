Hierarchy

Hierarchy is an important part of the website, every website follows a hierarchy for better resopnsiveness. But the hierarchy should also be viaible to the user so that the user can easily navigate through the website.

The Hierarchy can be shown by various method like breadcrumbs or a tree structure. Let us take the example of amazon website let us say we are looking for laptop so there is a tree structure in upper left to show how we can find a laptop and which are the parent pages above the current pages.
