Page Navigation

Page Navigation is an important part of web because it shows us that what webpage are we currently in from the list of the pages in the website. So user can easily understand the blue print of website and can easily find the data.

The page navigation must be a list of pages and with a selection of current page or breadcrumbs which show where are you exactly. Below you can see the illustrated images shows that the webpage shows that which page is user currently in and what are the other webpages in the website.
